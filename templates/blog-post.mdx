## Usage Instructions

### Step 1: Copy this template
Copy this file to `/content/posts/your-slug.mdx`

### Step 2: Update frontmatter
- Change `title`, `date`, `description`, `tags`, `slug` to match your post
- Set `published: false` while drafting, `published: true` when ready to publish
- The `slug` should match your filename

### Step 3: Register in blogs.ts
Add an entry to `src/data/blogs.ts`:
```typescript
{ slug: "your-slug", published: false },
```

### Step 4: Write your content
- Use standard markdown for most content
- Import and use special components as needed (HandwrittenNotesPDF, References, Cite)
- You can add custom React components if needed

### Step 5: Publish
1. Set `published: true` in frontmatter
2. Update `published: true` in `src/data/blogs.ts`
3. Optionally, set `BLOG_COMING_SOON = false` in `src/data/blogs.ts` if it's your first post
